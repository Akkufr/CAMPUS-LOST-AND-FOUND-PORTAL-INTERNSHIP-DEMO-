# Campus Lost and Found Portal

A modern web-based Lost & Found Management System designed to help students and staff report, search, and recover lost belongings efficiently within a college campus.

# 📖 Overview

The **Campus Lost and Found Portal** is a modern web-based application developed to simplify the process of reporting, searching, and recovering lost belongings within a college campus. Traditional methods such as notice boards, verbal communication, and manual record keeping often result in delays, confusion, and misplaced information. This project provides a centralized digital platform where students and staff can report lost or found items, browse available reports, search efficiently, and facilitate the return of belongings.

Built using **React**, **TypeScript**, **Vite**, and **Tailwind CSS**, the application delivers a fast, responsive, and intuitive user experience while demonstrating modern frontend development practices.

---

# 🎯 Problem Statement

Educational institutions frequently face challenges in managing lost and found items. Students often lose valuables such as ID cards, wallets, keys, mobile phones, calculators, USB drives, books, and accessories. Existing recovery methods rely heavily on notice boards or personal communication, making the recovery process slow and unreliable.

The Campus Lost and Found Portal solves this problem by providing a centralized system that allows users to report, search, and manage lost and found items digitally.

# ⚙ Installation

### Clone the Repository

```bash
git clone https://github.com/Akkufr/CAMPUS-LOST-AND-FOUND-PORTAL-INTERNSHIP-DEMO-.git
```

### Navigate into the Project

```bash
cd CAMPUS-LOST-AND-FOUND-PORTAL-INTERNSHIP-DEMO-
```

### Install Dependencies

```bash

```

### Start Development Server

```bash
npm run dev

```

# 📋 Usage Guide

### Reporting a Lost Item

1. Open the application.
2. Navigate to the Lost Item section.
3. Enter the item details.
4. Upload an image (optional).
5. Submit the report.

### Reporting a Found Item

1. Open the Found Item section.
2. Fill in the required details.
3. Upload an image.
4. Submit the report.

### Searching for Items

Users can search using:

- Item Name
- Category
- Keywords
- Location

### Claiming Items

Recovered items can be marked as **Claimed** to keep records organized and prevent duplicate reports.
